An AI-powered market intelligence platform built for ET AI Hackathon 2026.

## Structure

- `/ml` - Machine learning pipelines, data extraction, pattern intelligence, and LLM orchestration.
- `/backend` - API services and business logic (connecting ML to the frontend).
- `/frontend` - User interface for retail investors.
- `/docs` - Project documentation and SRS.
- `/scripts` - Utilities for running the application.
